# Terminal Journal

A minimal terminal-based journal project for writing, organizing, and reviewing notes from the command line.

## Project Goal

Terminal Journal is intended to be a small productivity tool for fast daily logging without opening a full notes app. It can grow into a focused command-line workflow for timestamped entries, search, tags, and exports.

## Planned Features

- Create dated journal entries from the terminal.
- Keep entries in a simple local file format.
- Search entries by date, keyword, or tag.
- Export entries to Markdown for backup or review.
- Keep the workflow lightweight and beginner-friendly.

## Suggested Roadmap

1. Add a basic CLI command for creating an entry.
2. Store entries in a `journal/` folder using Markdown files.
3. Add list, search, and tag commands.
4. Add tests for file creation and search behavior.
5. Add packaging instructions once the CLI is stable.
